# Simple Accordion Menu

A basic accordion menu built with plain **HTML**, **CSS**, and **JavaScript**.  
Click on each item to expand or collapse its content.

## ✨ Features

- No libraries or frameworks
- Pure vanilla JavaScript
- Simple and clean UI
